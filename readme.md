# 🚀 Resume Matcher — AI-Powered ATS Optimization Tool

An AI-powered Resume Matcher that analyzes resumes against job descriptions to improve ATS (Applicant Tracking System) compatibility. It provides semantic match scoring, identifies missing skills, generates AI-powered resume improvements, and helps candidates prepare for job applications.

## 🌐 Live Demo

🔗 **Try it here:** https://resume-matcher-ai-powered-ats.onrender.com/

---

## ✨ Features

- 📄 Upload resumes in **PDF** or **DOCX**
- 📝 Parse job descriptions
- 🎯 ATS Match Score
- 🔍 Skill Gap Analysis
- 🤖 AI-Powered Resume Rewriting
- 💼 Recruiter Outreach Message Generator
- 📧 Cold Email Generator
- 💰 Salary Estimation
- 🎤 AI Interview Question Generator

---

## 🛠 Tech Stack

- **Frontend:** Streamlit
- **Backend:** Python
- **AI Model:** Google Gemini API
- **Libraries:**
  - PyPDF2
  - python-docx
  - ReportLab
- **NLP:** Semantic Similarity & Text Analysis
